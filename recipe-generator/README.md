# AI Recipe Generator (Groq + Llama 3)

A simple command-line tool that generates clean, structured cooking recipes using Groq’s Llama 3 models.  
Enter any dish name (e.g., “pizza”, “pasta”, “omelette”) and the script outputs a short recipe and saves it as a Markdown file.

## Features
- Uses Groq’s ultra-fast Llama 3 models
- Generates a structured recipe with ingredients and steps
- Saves output into the `outputs/` folder
- Fully local CLI workflow

## Requirements
- Python 3.9+
- A Groq API key (free to generate)
